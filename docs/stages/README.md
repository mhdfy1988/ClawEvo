# Stages 文档说明

这个目录按阶段保存 TODO、状态和总结。

放在这里的文档类型只有三类：
- `stage-N-todo`
- `stage-N-status`
- `stage-N-report` / `stage-N-prework`

约定：
- 阶段内任务跟踪优先写在 `stage-N-todo`
- 阶段完成后，状态收口到 `stage-N-status`
- 总结和复盘收口到 `stage-N-*-report`
